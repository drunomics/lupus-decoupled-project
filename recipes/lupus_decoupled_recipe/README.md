# Lupus Decoupled Drupal Project Recipe

A Drupal recipe to setup the base project. This extends our
[Lupus Decoupled Drupal Recipe](https://www.drupal.org/project/lupus_decoupled_recipe/) to additionally install the
[Drupal CMS Starter](https://www.drupal.org/project/drupal_cms_starter) recipe and the`services_env_parameter` module.
